Clerk Guide: Applicant Registration Process (Section 3)

  1. Login to the System
    • Access the staff portal.
    • Enter your assigned email and password.
    • Keep credentials confidential.
  2. Navigate to 'Register Applicant'
    • Select "Register Applicant" from the dashboard.
    • Access the registration interface.
  3. Enter Applicant Information
    • Input all required details accurately.
    • Ensure completeness to avoid delays.
  4. Submit the Application
    • Verify information and click "Submit".
    • Correct errors if necessary before resubmitting.

Administrator Guide: Applicant Review and Approval Process (Section 4)

  1. Login to the System
    • Access the admin portal with your credentials.
    • Keep credentials secure.
  2. Navigate to 'View Applications'
    • Select "View Applications" from the dashboard.
    • View submitted applications.
  3. Select an Applicant
    • Choose an applicant and click "Review Applicant".
  4. Review Applicant Information
    • Examine all details and documents.
    • Verify accuracy and eligibility.
  5. Reject or Approve the Application
    • Approve if criteria are met.
    • Reject if requirements are not met, providing reasons.
