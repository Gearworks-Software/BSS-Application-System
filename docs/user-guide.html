Clerk Guide: Applicant Registration Process

  1. Login to the System
    • Access the staff portal.
    • Enter your assigned email and password.
    • Keep credentials confidential.
  2. Navigate to 'Register Applicant'
    • Select "Register Applicant" from the dashboard.
    • Access the registration interface.
  3. Enter Applicant Information
    • Input all required details accurately.
    • Ensure completeness to avoid delays.
  4. Capture Applicant's Document
    • Ensure the camera is active and functional.
    • Click "Scan Document" to capture the applicant's document.
    • Preview the captured image and ensure clarity before proceeding.
  5. Submit the Application
    • Verify information and click "Submit".
    • Correct errors if necessary before resubmitting.

Administrator Guide: Applicant Review and Approval Process

  1. Login to the System
    • Access the admin portal with your credentials.
    • Keep credentials secure.
  2. Navigate to 'View Applications'
    • Select "View Applications" from the dashboard.
    • View submitted applications.
  3. Select an Applicant
    • Choose an applicant and click "Review Applicant".
  4. Review Applicant Information
    • Examine all details and documents.
    • Verify accuracy and eligibility.
  5. Reject or Approve the Application
    • Approve if criteria are met.
    • Reject if requirements are not met, providing reasons.
  6. Navigate Between Pages
    • Use the navigation buttons to move between different sections of the application.
    • Click "Back" to return to the previous page or "Home" to return to the dashboard.
  7. Update Application Status
    • Approve or reject applications based on the provided details and documents.
    • Provide reasons for rejection if applicable.
    • Ensure the status is updated in the system for tracking purposes.

System Features

  • Camera Integration: Capture and preview applicant documents directly within the application.
  • Dynamic Navigation: Seamlessly navigate between pages using the application's stack-based navigation system.
  • Real-Time Updates: View and manage application statuses with instant feedback.
  • Theme Toggle: Switch between light and dark themes for better usability.
